Patent Dataset Analytics
This patent dataset consists of numerous patents (or observations, or samples) as rows and columns, as details (or measurements) for each observation such as 
application, published, priority dates, origin and filing jurisdictions, title, inventors etc.

Objectives are as follows:

Calculate the duration from application to publication by patent type.
Visualize jurisdiction and origin country patterns.
Build KNN models to predict duration and jurisdiction.
Provide a simple GUI to query predictions.
